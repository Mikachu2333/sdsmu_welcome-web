校园网

无线连接

  1. 新生账号激活：
    1. 到校报到完成后，连接名称为“sdsmu-net”的网络
    2. 在浏览器中打开https://slzfw.sdsmu.edu.cn:8800㊕
    3. 点击“用户激活”，账号为身份证号
    4. 按照流程完成设置密码[2]等一系列步骤即可激活账号
  2. 登陆：在浏览器中打开https://slrz.sdsmu.edu.cn㊕或http://210.44.80.65㊕并登陆即可
  3. 忘记密码：在浏览器中打开https://slzfw.sdsmu.edu.cn:8800㊕，点击右下角“忘记密码”并按提示进行重置，如果因更换手机号或手机号停机等原因导致无法重置，请联系班长或本班网络信息员
  4. 充值：点击https://slzfw.sdsmu.edu.cn:8800㊕，并登录；或在校园网的登陆页面点击“自助服务”按钮，登陆后即可使用支付宝充值（微信支付暂时无法使用）

    提示

    若点击“自助服务”按钮后网页显示“403 Error”，请将网址前缀“http”改为“https”，按回车即可。如发现无法正确弹出支付宝支付二维码（通常因校园网额度耗尽导致，说白了就是加载二维码需要流量，但你校园网流量用完了），请换用热点，并刷新该付款二维码页面

  5. 当前政策为每月免费 60G，超出部分按 0.5 ￥/G 收费，最多额外购买 60G 流量
  6. 警告：一个账号最多允许4台设备同时在线
  7. 严禁向他人共享校园网账号及一切校内资源

有线连接

提示

  1. 下面的说明为 Win10/Win11 教程，Mac 教程暂无
  2. 网线内部顺序从左到右应为“白橙橙，白绿蓝，白蓝绿，白棕棕”
  1. 按照校园网无线连接的教程激活校园网
  2. 将网线插入宿舍内的“H3C”盒子的底部右侧的接口内并正确连接到电脑
    1. 首次连接：
      1. 打开“设置”→“网络和 Internet”→“拨号”→“设置新连接”
      2. →“连接到 Internet”→ 点击“否，创建新连接(C)”→ 选择“宽带(PPPoE)(R)”
      3. → 输入自己的校园网帐号以及密码 → 勾选“记住密码”
    2. 再次连接：
      1. 打开“设置”→“网络和 Internet”→“拨号”
      2. → 选中之前设置的网络 → 连接即可

空调使用教程

重要

  1. 空调按使用时长收费，不按电费收费。具体收费及租赁政策详见“海享租”公众号
  2. 请各位同学在搬离校区或毕业前退租空调，否则将导致下一届学弟学妹无法租赁！

  4. → 点击“设备”→“空调图标”→“成员管理”，在此页面下将宿舍全部成员权限设置为均可管理空调开关即可

搜索设备提示

搜索不到设备请务必开启“蓝牙”功能，学校的设备无法扫码连接

    1. 在手机应用市场下载“大白 U 帮”app
    2. 按照实际住宿情况注册
    3. 授予并开启“定位”与“蓝牙”权限
  2. 本楼层小浴室使用：
    1. 带好洗浴物品前往公共厕所旁边的浴室排队
    2. 进入浴室，点击如右图所示的按钮 → 选择“蓝牙设备”→“点击进行时”
    3. →“洗澡”→“搜索洗澡”
    4. → 选择设备[5]→“开始洗澡”
    5. 结束后点击“结束洗澡”按钮，并结算
  3. 一层公共大浴室预约：
    1. 在软件初始界面根据实际情况选择“X 号楼 1 层”的浴室
    2. → 点击一个浴位，并点击“预约”按钮（若已满请选择“排队”）

    4. → 结束后点击“结束洗澡”按钮，并结算

  5. 申诉：如果在洗澡时突然停电导致无法结束洗澡而被扣费，请按照软件打开时弹出的公告，联系相关工作人员处理

洗衣机/洗鞋机使用教程

重要

  1. 禁止向第二格内倾倒洗衣粉、洗衣液，第二格是放柔顺剂的！
  2. 禁止使用洗衣机洗鞋，请用旁边的洗鞋机！
  3. 禁止在洗衣机上堆放杂物
  4. 请尽量手洗袜子、内衣内裤等贴身衣物

浮烟山校区

  1. 在微信小程序搜索“海乐生活”，并注册、开启相机与定位权限
  2. 扫码直接使用：
    1. 前往洗衣机 → 在小程序内点击“扫码使用”按钮
    2. → 扫描洗衣机上的二维码 → 选择并下单
    3. → 放入衣物与洗衣粉/洗衣液，输入验证码后缴费即可
  3. 收费标准与营业时间详见软件说明

虞河校区

  1. 微信小程序搜索“智慧笑联”→ 按照提示注册并授予定位权限
  2. → 绑定“潍坊医学院（虞河校区）”→ 绑定至自己所在的宿舍楼
  3. →在小程序内再次扫描洗衣机上的二维码，按提示操作并付款即可
  4. 收费标准与营业时间详见软件说明

洗衣机错误处理办法（若无相关经验切忌自行操作）

  1. 拨打洗衣机旁边的报修电话
  2. E1：洗衣机断电后开门，打开洗衣机右下角小门，旋开阀门，使用镊子等工具伸入并清除其中堵塞管道的杂物，恢复原样即可
  3. E4：旋开洗衣机后方的水管阀门即可，若仍无反应说明水压过低，请向宿管反应

使用前后务必控干水箱并清理滤网

请假流程

注

  1. 需在学校统一将大家拉入钉钉的“山东第二医科大学”企业后方可使用
  2. 各学院要求不一，仅以临床医学院为例
  3. 病假须有校医院或二级甲等及以上医院开具的相关证明；事假需提供证明，由所在学院核实情况后审批

重要

请大家自觉销假，切忌一再拖延

线下请假步骤（一般情况）

  1. 前往学工办或班主任办公室，当面请假并获得假条
  2. → 根据老师要求扫描相关二维码 → 钉钉填表
  3. → 刷脸进出校门，并将请假条之一交给保卫处
  4. → 返校后，在钉钉的电子假条处，以评论的方式销假

  1. 打开钉钉 → 点击左上角选择主企业为“山东第二医科大学”
  2. → 点击页面最下方菜单栏“工作台”→ 点击“OA 审批”
  3. → 在“学生日常事务管理”类选择“浮烟山校区本科学生请假单”→ 填表
  4. → 电话联系班主任老师或学工办老师，说明请假事由并等待审批
  5. → 审批通过后刷脸进出校门
  6. → 返校后，在钉钉的电子假条处，自觉以评论的方式销假

走读办理流程

提示

本节内容参考自《山东第二医科大学校外走读办理说明》，部分内容可能随政策变动而更改，请以实际情况为准

注意

大一新生严禁在外住宿，任何人未经允许严禁擅自在外住宿。此外，本节的所有申请表等类似文件有效期均不超过一年且不可跨学年

首次办理

  1. 填写《校外走读申请表》（下简称“申请表”），一式三份
    1. 学生需填写的信息：个人信息、理由、个人承诺等，需落款时间、签名并按手印
    2. 学生父母其中一方需填写的信息：外宿（走读）理由、相关情况，个人信息、知晓情况、承诺等，同样需落款时间、签名并按手印，父母需到校填表，无法到校的必须与辅导员视频通话说明情况，视频由学工办留存备案
  2. 填写《山东第二医科大学学生校外住宿安全保证书》，一式三份，要求同“申请表”
  3. 提供学生及其父母的身份证复印件，并根据要求书写落款时间与保证、签名并按手印
  4. 填写《山东第二医科大学本科生退宿申请表》，一式三份
  5. 此外应按学工办要求与班长、舍长报备，其他要求以学工办为准

续办

  1. 填写《山东第二医科大学校外走读申请表（续表）》，一式三份，要求同上述“申请表”
  2. 其他要求同上文所述，略

浮烟山校区

宿舍维修

  1. 加入各宿舍楼的 QQ 报修群，在群内反映具体故障
  2. 前往一层宿管处填表报修
  3. 在宿舍一层宿管旁边的公告栏处查看相关负责人的电话，直接拨打即可
  4. 拨打学生公寓管理中心电话反馈
  5. 询问带班学长、学姐

教室维修

  1. 拨打后勤管理处的电话报修
  2. 拨打教室管理中心的电话报修（仅限多媒体及饮水机）
  3. 拨打物业电话报修
  4. 在“诉求留言”微信小程序内反馈

虞河校区

宿舍维修

在宿管处填写报修单据并送往物业维修中心（位置见虞河校区常用地点表）

教室维修

拨打教室内张贴的报修电话

教学楼多媒体教室/乐道济世书院第二课堂申请流程

  1. 提前做好《活动策划案》、《活动安全应急预案》
  2. 请指导老师（通常为班主任或学工办老师）审核上述文件并确认钉钉具体审批负责人
  3. 打开钉钉 → 点击左上角选择主企业为“山东第二医科大学”
  4. → 点击页面最下方菜单栏“工作台”→ 点击“OA 审批”
  5. → 在“校园文化活动与社会实践”类选择对应申请表并填写（或直接搜索“教室”/“书院”）
  6. 前往教室 E 区 2 层（见浮烟山校区教学楼地图）靠近 A 区处的“教室管理中心”签字盖章
  7. 前往预约的教室，拨打讲台上教室管理员的电话提前沟通说明

文体中心预约教程

提示

场馆开放时间见此

  1. 关注“山二医文体中心”公众号 → 点击“场地预约”→“预约入口”
  2. → 点击“我的”→“校内登录”→ 使用 CAS 认证系统的账号密码登录
  3. 进入个人中心页面，点击“人脸录入”，录入信息 →
  4. 按需预约游泳馆、羽毛球馆等，并付费 →
  5. 在预定时间段前往场馆，向工作人员出示二维码即可

费用（校内学生）

提示

因羽毛球馆和篮球馆共用同一场地，故此二者互斥；场馆具体开放时间以及价格变动以公众号通知为准

注

  1. 羽毛球等多人运动项目预约方式为“单人预约，多人共享”，详情可咨询负责相关场馆的教师
  2. 游泳馆仅限本人购票，代购无效，每人每场限购一张
  1. 健体中心：3 元/2 小时
  2. 羽毛球馆：6 元/时/片 3 元/人/2 小时
  3. 篮球馆：20 元/1 小时（半场）或 40 元/1 小时（全场）
  4. 游泳馆：8 元/场/2 小时
  5. 室内乒乓球场：2 元/台/2 小时
  6. 室内网球场：6 元/片/1 小时

医保

  1. 门诊医疗费：需在校医院内就诊方可使用医保，支付比例为 65％，一个医疗年度内最高支付限额为 450 元，医保报销相关政策详询校医院或参考《学生手册》相关章节内容
  2. 参保学生需临时在外地就医的，医保报销相关政策详询校医院或参考《学生手册》相关章节内容
  3. 住院医疗费：统一按低档缴费，享受高档待遇。详细政策敬请咨询校医院或参考《学生手册》相关章节内容
