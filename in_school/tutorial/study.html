高校 edu 教育邮箱

重要

  1. 严禁出借、买卖邮箱，这是违规的！
  2. 禁止交流机密信息、外借、泄露信息或参与违反法律法规或相关规章制度的活动
  3. 每 60 天必须更改一次密码（密码建议 14 ～ 16 位，含大小写、数字、特殊字符如下划线等）

简要说明

  1. 在校生可免费开通以 @stu.sdsmu.edu.cn 结尾的教育邮箱（也称学生邮箱、校园邮箱），毕业时回收
  2. 主要用于：学术交流、投递简历、向科技公司申请学生优惠等

申请流程

  1. 打开“山东第二医科大学”app→ 点击底部菜单“应用”
  2. 点击“业务申请”→“学生邮箱申请流程”→“我要办理”
  3. 按照要求填写所有信息并仔细阅读说明 → 提交信息，等待通过
  4. 提交后的下个月 1 号自动注册，进入下一小节的首次登录教程

注意

以下操作必须使用电脑端浏览器进行

首次登录激活

  1. 在提交信息后的下个月第一天后（推荐 2 ～ 9 号）方可进行此小节步骤，账号创建 10 天后逾期未登录视为放弃账号申请

  2. → 打开https://edu.icoremail.net → 点击“忘记密码”

  3. → 选择“山东第二医科大学”→ 按照步骤重置密码（相当于首次设置密码）

    密码强度

    建议 14 ～ 16 位，含大小写、数字、特殊字符如下划线等

  4. → 再次打开https://edu.icoremail.net（或https://mail.stu.sdsmu.edu.cn），并登录

设置

注意

本小节的所有操作必须使用电脑端浏览器进行

常规设置

关闭“反垃圾邮件”[1]功能
  1. 在邮件主页面单击最左下角的“设置”→ 安全设置 →
  2. “反垃圾机制”→ 调整选项为“关闭”
设置发信人名称（根据需要设置）
  1. 打开左下角“设置”→“收发信设置”→
  2. “写信设置”→ 勾选或取消“发件人显示我的姓名：×××”选项即可
将邮箱账户关联至“山东第二医科大学”app
  1. 打开安全中心→ 点击上方菜单“账户安全”→“安全设置”→
  2. 点击“绑定/解绑邮箱”→ 按要求填写要绑定的校园邮箱 → 确认后刷新页面
  3. 等待几天后再使用学校 app 即可正常查看邮件

安全设置

开启二次验证
  1. 点击左下角“设置”→“安全设置”→ 切换至“二次验证设置”→
  2. 推荐同时开启“短信”、“备用邮箱”、“微信”三项，按提示直接操作即可 →
  3. 如果了解 OTP 并知道如何开启、如何调整可以实验性地开启此选项
设置备用邮箱

与二次验证在同一页面，略

生成“客户端专用密码”
  1. 本密码用于开启二次验证后，使用其他设备接收、发送邮件；各软件各设备应使用不同的客户端密码
  2. 点击左下角“设置”→“安全设置”→ 切换至“客户端专用密码”→
  3. 点击“生成专用密码”→ 输入备注名称 → 保存生成的密码（只显示一次，请务必牢记）
配置第三方邮件客户端（以 Thunderbird 为例）

注

以 Thunderbird 为例的原因是该软件应用较广泛且配置过程最为繁琐。一般用户推荐使用 QQ 邮箱、网易邮箱大师等成熟、广泛使用的客户端以减少折腾的麻烦程度。通常国内的邮箱客户端大都可以自动应用设置而无须手动操作，若自动配置失败也可参照本节内容

特别说明

邮箱网站给出的相关服务器地址是错误的，请忽略它的所有提示信息并按照下文给出的信息重新填写

  1. 首先，参照上方“生成‘客户端专用密码’”小节的内容，将生成的密码保存备用
  2. 切换 Thunderbird 至“账户管理”页面，点击“添加邮件账户”→
  3. 输入账号、专用密码 → 点击左下角“高级”→ 根据下文内容逐一对应填写即可

空闲教室查询

提示

在临近期末考试时，因考试教室占用等原因查询结果可能不准确

  1. 根据 CAS 认证系统登录教程，登录山二医 app（下载链接见此）
  2. 点击下方菜单栏“应用”→ 空闲教室查询
  3. 点击右上角“〉”按钮 → 根据自己的需求进行筛选

图书馆座位预约教程

  1. 微信小程序搜索“青栀校园”→ 微信注册登录并绑定学号 → 允许小程序通知
  2. → 在小程序内点击“座位预约”或扫描图书馆座位上的二维码即可

暂离座位与注意事项

  1. 如因各种原因需要长时间离开的，请在小程序上选择“暂时离开”，否则按违规处理
  2. 离馆时也需要在小程序内确认
  3. 如发现已预约的座位被他人占据，请扫描桌面上的二维码并在小程序内举报，工作人员将尽快处理

违规说明

  1. 已预约而未按时到位的记一次违规
  2. 未选择暂离而离开座位被举报的记一次违规
  3. 停止使用后未选择退馆的记一次违规
  4. 三次违规后将取消座位预约资格三天

其他禁止事项

  1. 禁止占用其他人已经预约的座位
  2. 禁止在图书馆内喧哗、吸烟
  3. 禁止在图书馆的非背诵区域内背诵、朗诵、频繁交流
  4. 禁止在图书馆内谈恋爱、亲嘴
  5. 禁止在图书馆占座，馆内每天清书

CAS 资源访问控制系统（校内 VPN）

提示

  1. 在校内时可通过校园网直接访问相应系统，无需使用本系统中转
  2. 新生入学后 3 周左右，信息录入完毕方可使用

  1. 官网：https://webvpn.sdsmu.edu.cn

  2. 说明：也称 CAS 认证系统、智慧校园系统，本系统主要用于在校外访问校内网络信息资源，如：教务系统（查成绩、课表）、知网[2]、临床医学虚拟仿真实验中心等

  3. 异地登录教务系统教程（其它系统同理）：

    1. 打开网站，点击“统一身份认证登录”，使用学号+密码登录或手机号验证登陆、扫描登陆等方式均可（推荐绑定微信，初始密码为 sdsmu@身份证后六位）

    2. → 找到应用中心 →“教务系统——非单点登录”→ 使用教务系统账号密码登录即可

      注意

      请注意，在校外时点击“教务系统”无法登录，只有“非单点”能校外登录！

CARSI 系统

  1. 官网：https://ds.carsi.edu.cn
  2. 说明：（与 CAS 认证系统作用不同）用于快速访问学校订阅的各类数据库、查询文献等，如百度文库、知网、万方、维普等
  3. 使用教程：
    1. 进入官网 → 搜索“山东第二医科大学”并勾选“记住我的选择”→ 点击后进入登陆界面

      提示

      请不要收藏登录界面的网址！每次登录网址都不一样，只能从官网重新进入

    2. → 使用CAS 认证系统的账号密码登录系统，出现各类弹窗一律选择“Accept”即可

      注意

      仅推荐在自己的电脑上如此设置，如必须在网吧等公共场所的电脑上使用，请审慎阅读相关提示，并谨慎进行登录，如因账号泄露造成损失，一切责任自负

    3. → 登录完成后，点击任意资源链接即可进入相应网站并获取论文

校务行（微信小程序）

提示

新生可能无法在开学后即刻使用本程序，需等待学校将信息录入完毕

  1. 官网：微信小程序“校务行”
  2. 用途：查成绩，下载学籍证明、成绩证明的 pdf 版本
  3. 费用：以程序显示为准
  4. 教程：
    1. → 点击小程序右上角“点击登录”（帐号为学号，密码为身份证后六位）
    2. → 按需选择“电子成绩单”或“电子证明”→ 按照小程序提示进行即可

缓考申请教程

注意

各学院要求不一：临床医学院大部分课程可以只填写钉钉、教务系统，仅少数要求同时提交纸质表格；其他学院以教师要求为准

  1. 填写钉钉[3]（详细步骤如下）
    1. 打开钉钉 → 点击左上角选择主企业为“山东第二医科大学”
    2. → 点击页面最下方菜单栏“工作台”→ 点击“OA 审批”
    3. → 在“学风建设”类选择“学生缓考审批表”（或直接搜索“缓考”）
  2. 在教务处下载《山东第二医科大学学生缓考审批表》，打印 3 份并按照要求填写完毕
  3. 填写教务系统（详细步骤如下）
    1. 进入教务系统（仅校园网，详情见此）
    2. 点击左侧菜单“考试报名”→“我的申请”→“缓考申请”
    3. → 选择“学年学期”和“活动名称”后，直接点击“搜索”（不要填写科目名称）
    4. → 在弹出的菜单中选择缓考科目并填写申请[4]
  4. 前往学工办交表并等待审批

学工系统（微信小程序）

  1. 用途：学工系统主要用于晚点名、返校信息填报等日常工作[5]
  2. 使用方式：
    1. 打开“定位权限”并允许微信使用 → 在微信搜索“智慧学工”小程序
    2. 根据学校下发的账号密码进行登录（推荐立即与微信绑定以免忘记密码）
